# Week 4 Lab: Verification Toolchain

- A registry is a collection of information about packages.
- A package is a set of files meant to be used in a project.
- A 'normal' library is meant to produce a file (.a/.so) in the memory to be referenced. An interface library is a method to package include directories or compiler flags so other targets may use their definitions/properties during link time.
- A top module is the highest module in a hardware design. It is the container that links all of the sub-modules and represents the final system being designed.
